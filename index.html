Algorithm Visualizer

Spring 2023 | C++ | Github repository

Application for simulating pathfinding and sorting algorithms.

* Graphics engine used : SFML.

PathFinding

You can place walls and obstacles. Then you can find the shortest path between the start and end node with different algorithms. You can generate a random maze using the recursive division algorithm.

Algorithms implemented:

  • Dijkstra
  • A*
  • BreadthFirst
  • DepthFirst
  • Bidirectional

Sorting

A sequence of numbers are generated randomly. Then you can sort it with different algorithms and visualize it, going forward and backward on the simulation.

Algorithms implemented:

  • BubbleSort
  • MergeSort
  • QuickSort
  • InsertionSort
  • SelectionSort
  • CocktailSort

Chess Engine with Alpha-Beta pruning

Winter 2024 | C++11 OpenGL | Github repository

Chess Engine with search function based on Negamax algorithm (similar to minimax).

Tentative game strength level : 1800 ELO (Amateur player).

* Custom graphics Engine developed with OpenGL.

Implementation details :

  • Search enhanced with alpha-beta pruning.
  • Transposition table using zobrist hashing.
  • Quiescence search extension on captures.
  • Iterative Deepening
  • Materialistic evaluation.

Smartwatch on Samsung S3C44BOX

Winter 2024 | C EclipseARM | Github repository

Embedded software university project, Smartwatch developed in Samsung S3C44BOX microcontroller.

Features :

  • Fixed-point arithmetic for statistical calculations.
  • Multi-threading (cooperative scheduling).
  • Audio recording and playback.
  • LCD screen and Keypad for user input.
  • Track for 7 statistics like velocity, calories and steps.

Audio menu :

User can record audio as voice notes. They are stored in memory and can be reproduced later at normal speed and double speed.

  • Audio recording and playback.
  • Storage for 3 voice notes
  • Play at normal and x2 speed

Statistics menu :

We count the steps and heartbeats that occur in 0.5 second intervals. Using the results from the last 10 seconds (20 samples) we calculate the average steps and heartbeats per minute. The rest of the statistics are derived from these averages. Some of these are decimal numbers, so we use Q22.10 fixed-point representation and perform calculations using fixed-point arithmetic.

  • User select his three favorites statistics
  • Fixed-point arithmetic for statistical calculations.
  • Velocity
  • Steps
  • Heart beats
  • Calories

Conway’s Game Of Life

Summer 2023 | Java | Github repository

Implementation of cellular automata with statistics and different game variants. Utilized MVC, observer, and factory patterns.

Implementation details :

  • Factory design pattern
